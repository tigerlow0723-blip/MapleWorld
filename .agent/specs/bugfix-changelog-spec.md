## 버그 수정 이력

### 1. RNG 결정론적 패턴 제거
- **커밋**: `f9e0496`
- **문제**: 보상/상점 아이템 생성 시 동일한 랜덤 패턴이 반복됨
- **조치**: `GameManager`에 `clientRandomSeed` 및 `ReseedClientRandom` 도입
  - 유저 ID + 경과 시간 + 엔티티 ID + reseed 카운트를 해싱하여 시드 생성
  - 게임 시작, 상점 열기, 보상 생성 등 주요 분기점에서 reseed 호출

### 2. 아이템 사람마다 동일하게 뜨는 오류
- **커밋**: `0ac2325`
- **문제**: 아이템 드롭/상점 아이템이 모든 플레이어에게 동일하게 생성
- **조치**: 클라이언트별 독립 RNG 시드 적용

### 3. 몬스터 공격 초기화 오류
- **커밋**: `b154f1d`
- **문제**: 몬스터 턴 시작 시 공격 관련 상태가 초기화되지 않음
- **조치**: `ResetWeaponAttackCounts`, `ResetArmorActivationCounts` 호출 추가

### 4. 인벤토리 셀 복사 버그
- **커밋**: `402f2c7`
- **문제**: 인벤토리 셀이 의도치 않게 복사됨
- **조치**: 아이템 배치 로직 중복 방지 처리

### 5. 수동 판매 수정
- **커밋**: `56db4cc`
- **문제**: 수동 판매 시 가격 계산 오류
- **조치**: 금괴/다이아몬드 수동 판매 가격 고정

### 6. 보스 3페이즈 사망 모션
- **커밋**: `3a0c1f8`
- **문제**: 보스 마지막 페이즈 사망 시 애니메이션 미재생
- **조치**: `isDyingAnimating` 플래그 추가, DIE2 상태 전용 애니메이션 재생

### 7. 노가다 목장갑 업그레이드 후 형태 오류
- **커밋**: `9d191ca`
- **문제**: 업그레이드 후 아이템의 shapeMask가 갱신되지 않음
- **조치**: 업그레이드 시 shapeMask 재계산 로직 추가

### 8. UI 위치 변경 대응
- **커밋**: `0ac2325`, `d6d90cb`
- **문제**: 출시 후 MSW 플랫폼 업데이트로 UI 위치가 변경됨
- **조치**: UI 엔티티 위치/크기 재조정

### 9. 방어구 시너지 미리 적용 방지
- **커밋**: `19ecdbc`
- **문제**: 전투 시작 전에 방어구 시너지가 미리 적용됨
- **조치**: `ClearArmorTemporaryEffects` 호출 타이밍 수정
